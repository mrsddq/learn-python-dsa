# Python DSA Learning Roadmap

## Phase 1: Python Fundamentals

- variables and data types
- conditionals and loops
- functions
- lists, tuples, dictionaries, and sets
- file handling
- basic testing with `pytest`

## Phase 2: Core Complexity

- Big O notation
- time vs space tradeoffs
- recursion stack cost
- amortized analysis for dynamic arrays

## Phase 3: Linear Data Structures

- arrays and lists
- linked lists
- stacks
- queues
- deques
- hash maps

## Phase 4: Searching and Sorting

- linear search
- binary search
- selection sort
- insertion sort
- merge sort
- quicksort

## Phase 5: Recursion and Backtracking

- recursion fundamentals
- first/last index patterns
- subsets and permutations
- backtracking decision trees

## Phase 6: Trees and Graphs

- binary trees
- binary search trees
- BFS and DFS
- graph representations
- shortest paths

## Phase 7: Dynamic Programming

- memoization
- tabulation
- knapsack patterns
- sequence DP
- grid DP

## Weekly Practice Loop

- solve 5 small problems
- rewrite 2 old solutions cleanly
- add tests for 1 topic
- document complexity for every new reference solution

| Linear search | O(n) | O(1) |
| Binary search | O(log n) | O(1) |
| Merge sort | O(n log n) | O(n) |
| Stack push/pop | O(1) | O(n) |
| Queue enqueue/dequeue | O(1) amortized | O(n) |
